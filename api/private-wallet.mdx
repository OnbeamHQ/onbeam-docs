Base URL: `https://api.onbeam.io`

Private wallets are isolated accounts intended for higher-privacy or compliance-sensitive flows. They typically have stricter access controls and auditability.

## Create private wallet

Method: `POST`  
Path: `/private/wallets`

Creates a new private wallet.

### Request body

- `owner` (string, required): Address that controls the private wallet.
- `label` (string, optional): Human-readable label for the wallet.
- `tags` (array, optional): List of tags for categorization, for example `["vip", "compliance"]`.

### Response

- `walletId` (string): Private wallet identifier.
- `address` (string): Wallet address.
- `createdAt` (string): Creation timestamp.

## List private wallets

Method: `GET`  
Path: `/private/wallets`

Returns a paginated list of private wallets.

### Query parameters

- `cursor` (string, optional)
- `limit` (integer, optional)
- `tag` (string, optional): Filter by a specific tag.

### Response

- `data` (array): List of private wallet objects.
- `cursor` (string): Cursor for pagination.

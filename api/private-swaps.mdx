Base URL: `https://api.onbeam.io`

Private swaps provide an additional layer of isolation and control. They can be tied to private wallets or restricted routing policies.

## Create private swap

Method: `POST`  
Path: `/private/swaps`

Executes a private token swap using balances from a private wallet or restricted pool.

### Request body

- `fromPrivateWalletId` (string, required): Private wallet funding the swap.
- `fromChainId` (integer, required)
- `toChainId` (integer, required)
- `fromToken` (string, required)
- `toToken` (string, required)
- `amount` (string, required)
- `slippageBps` (integer, optional)
- `policyId` (string, optional): Identifier of a routing or compliance policy.

### Response

- `swapId` (string)
- `status` (string)
- `from` (object)
- `to` (object)

## Get private swap

Method: `GET`  
Path: `/private/swaps/{swapId}`

Fetch the latest status of a private swap.

### Path parameters

- `swapId` (string, required)

### Response

- `swapId` (string)
- `status` (string)
- `transactions` (array)
- `error` (object, nullable)

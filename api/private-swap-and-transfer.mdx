This API provides separate building blocks:

- Private balances: `GET /privacy/balances`
- Private swap: `POST /privacy/swap/quote` and `POST /privacy/swap/prepare`
- Private transfer: `POST /privacy/transfer`

Clients can orchestrate these steps depending on the UX (shield/unshield, swap, then transfer).

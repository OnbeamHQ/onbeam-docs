Base URL: `https://api.onbeam.io`

Unified balance endpoints provide a cross-chain view of assets, aggregating tokens and valuations across all supported networks.

## Get wallet unified balance

Method: `GET`  
Path: `/wallets/{walletId}/balance/unified`

Returns an aggregated balance view for a single wallet.

### Path parameters

- `walletId` (string, required): Wallet identifier or address.

### Query parameters

- `currency` (string, optional): Fiat currency code for pricing, for example `USD`.
- `includePerChain` (boolean, optional): If `true`, include per-chain breakdowns.

### Response

- `walletId` (string)
- `totals` (array): Aggregated token balances and valuations.
- `perChain` (array, optional): Per-chain token balances if requested.

## Get project aggregated balance

Method: `GET`  
Path: `/balances/unified`

Aggregates balances across all wallets for your project or API key.

### Query parameters

- `currency` (string, optional): Fiat currency for pricing.
- `groupBy` (string, optional): One of `token`, `wallet`, or `chain`.

### Response

- `totals` (array): Aggregated token balances and valuations.
- `groups` (array): Optional group breakdowns, depending on `groupBy`.

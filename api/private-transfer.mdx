Base URL: `https://api.onbeam.io`

Private transfers route funds from private wallets or under stricter policy controls, useful for compliance or higher-privacy flows.

## Create private transfer

Method: `POST`  
Path: `/private/transfers`

Creates a transfer funded by a private wallet.

### Request body

- `fromPrivateWalletId` (string, required): Private wallet funding the transfer.
- `toAddress` (string, required): Recipient address.
- `fromChainId` (integer, required)
- `toChainId` (integer, required)
- `token` (string, required)
- `amount` (string, required)
- `policyId` (string, optional): Identifier of a routing or compliance policy.

### Response

- `transferId` (string)
- `status` (string)
- `from` (object)
- `to` (object)

## Get private transfer

Method: `GET`  
Path: `/private/transfers/{transferId}`

Fetch details and status for a specific private transfer.

### Path parameters

- `transferId` (string, required)

### Response

- `transferId` (string)
- `status` (string)
- `transactions` (array)
- `error` (object, nullable)

# Balances

Auth:

- JWT only

## Get balances

```bash
curl "http://localhost:3004/balances?chains=1,137&includePrices=true" \
  -H "Authorization: Bearer $JWT"
```

Query params:

- `chains` (optional): comma-separated chain IDs (example: `1,137`)
- `includePrices` (optional): boolean

# Getting Started

## Run the API locally

```bash
npm install
npm run start:dev
```

Default base URL:

- `http://localhost:3004`

## Verify

- Swagger UI: `http://localhost:3004/api`
- OpenAPI JSON: `http://localhost:3004/api-json`

## Mintlify local docs (this repo)

Mintlify CLI requires Node 20+.

```bash
npx mintlify dev onbeam-docs
```
